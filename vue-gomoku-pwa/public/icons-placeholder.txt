# PWA图标文件占位符
# 
# 在实际部署中，需要将以下SVG图标转换为PNG格式：
# - icon-192x192.png (192x192像素)  
# - icon-512x512.png (512x512像素)
# - favicon.ico (32x32像素)
#
# 可以使用以下工具：
# 1. 在线转换工具：https://svgtopng.com/ 或 https://convertio.co/svg-png/
# 2. 命令行工具：librsvg-bin (Ubuntu) 或 inkscape
# 3. Node.js工具：sharp 库
#
# 转换命令示例：
# rsvg-convert -w 192 -h 192 icon.svg > icon-192x192.png  
# rsvg-convert -w 512 -h 512 icon.svg > icon-512x512.png
#
